# 🏢 Funes Departamentos - Sistema de Gestión

Sistema de gestión de alojamientos desarrollado para la administración de reservas, clientes y departamentos.

## 🚀 Descripción

Aplicación diseñada para gestionar la disponibilidad de departamentos en tiempo real, permitiendo registrar reservas y administrar clientes de manera centralizada.

El sistema fue pensado para facilitar la operación diaria, evitando conflictos de fechas y mejorando la organización de la información.

## ⚙️ Funcionalidades principales

- ABM de clientes  
- ABM de departamentos  
- Registro de reservas  
- Control de disponibilidad por fechas  
- Visualización de ocupación  

## 🧠 Tecnologías utilizadas

- Java  
- SQLite  
- Java Swing / interfaz gráfica
